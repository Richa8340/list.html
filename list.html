Common HTML Tags

  1. <html> - The root element that defines the entire HTML document. All other HTML elements must be nested inside this tag.
  2. <head> - Contains meta-information about the HTML document, such as the title, character set, and linked stylesheets or scripts.
  3. <title> - Sets the title of the HTML document, which appears in the browser's title bar or tab.
  4. <meta> - Provides metadata about the HTML document, such as character encoding, author, and viewport settings.
  5. <link> - Defines the relationship between the HTML document and an external resource, commonly used to link to stylesheets.
  6. <script> - Embeds or links to JavaScript code that can be executed in the browser.
  7. <body> - Contains the content of the HTML document that is displayed in the web browser, such as text, images, and links.
  8. <h1>, <h2>, <h3>, <h4>, <h5>, <h6> - Define headings of different levels, with <h1> being the highest and <h6> the lowest.
  9. <p> - Defines a paragraph of text.
  10. <a> - Defines a hyperlink that links to another webpage or resource. It uses the href attribute to specify the URL.
  11. <img> - Embeds an image into the document. It uses the src attribute to specify the image file and the alt attribute for alternative text.
  12. <ul> - Defines an unordered list of items, which are usually marked with bullet points.
  13. <ol> - Defines an ordered list of items, which are usually numbered.
  14. <li> - Defines a list item, used within <ul> or <ol> tags to represent each item in the list.
  15. <table> - Defines a table. It is used in conjunction with <tr>, <th>, and <td> tags to create rows, headers, and cells.
  16. <tr> - Defines a table row, used inside the <table> tag.
  17. <th> - Defines a table header cell, which is bold and centered by default.
  18. <td> - Defines a table data cell, which contains data in a table.
  19. <form> - Defines an HTML form used to collect user input. It can contain various input elements like text fields, checkboxes, and buttons.
  20. <input> - Defines an input field within a form, such as text, password, or checkbox.
  21. <button> - Defines a clickable button, which can be used to submit forms or trigger JavaScript functions.
  22. <div> - Defines a block-level container used to group and style content. It has no semantic meaning on its own.
  23. <span> - Defines an inline container used to group and style text or other inline elements. It has no semantic meaning on its own.
  24. <footer> - Defines a footer for a document or section, often containing information like copyright, contact details, or related links.
  25. <header> - Defines a header for a document or section, often containing introductory content, navigation links, or branding.
